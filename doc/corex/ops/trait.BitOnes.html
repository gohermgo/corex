Expand description

The type-agnostic ‘one’-ing operator

Examples

An implementation of Not for a signed type, hopefully can be enable the use of .+ to zero the bits

use corex::ops::BitOnes;

assert_eq!(i32::MAX.bitones(), -1i32);
assert_eq!(u32::MIN.bitones(), u32::MAX);
